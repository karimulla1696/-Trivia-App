# Trivia App

Node Version 10X, 
NPM Version 6.11.3, 
Angular Version 8.0.6

## First Step

```
clone the project in github link https://github.com/karimulla1696/Trivia.git
```

## Second Step
```
npm install
```

## Third Step
```
ng build
```

## Final Step
```
npm start
```

## Start the app in browser
```
http://localhost:9090/
```
